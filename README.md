# twitch-bot
A twitch.tv bot where users can play text games and grab statistics.

# Getting Started
```bash
$ git clone github.com/hueyjj/twitch-bot
```
or
```bash
$ go get -u github.com/hueyjj/twitch-bot
```


# Running
```bash
$ cd github.com/hueyjj/twitch-bot
$ make
$ ./bin/twitch-bot
```


# Cleaning
```bash
$ make clean
```
